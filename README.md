Salary Prediction and Professional Factor Analysis
This project explores the impact of various demographic and professional attributes on individual salary levels. By utilizing regression analysis and statistical modeling, the project aims to quantify how experience, age, and gender correlate with compensation across the dataset.

Project Files
3.ipynb: The Jupyter Notebook containing the full analytical pipeline, from data cleaning and outlier detection to model training and evaluation.

3.csv: The dataset used for the analysis, containing features such as ID, Experience Years, Age, Gender, and Salary.

3.png: A data visualization export showing the correlation between independent variables and salary outcomes.

Core Objectives
Correlation Analysis: Determining the strength of the relationship between years of experience and annual income.

Demographic Insights: Investigating salary distribution across different age groups and gender categories.

Regression Modeling: Building a predictive model to estimate potential salary based on professional profiles.

Data Integrity: Implementing robust preprocessing to handle high-variance salary data and ensure model accuracy.

Technical Implementation
Data Cleaning: Identification and treatment of anomalies in reported salary and experience figures.

Exploratory Data Analysis (EDA): Use of distribution plots and scatter matrices to understand the underlying data structure.

Libraries Used:

pandas (Data manipulation)

matplotlib and seaborn (Data visualization)

scikit-learn (Statistical modeling and regression)

How to Run
Installation:
Ensure you have a Python environment set up with the necessary libraries:

Bash
pip install pandas matplotlib seaborn scikit-learn
Execution:
Open the 3.ipynb notebook in any Jupyter environment (Lab, Notebook, or VS Code) and execute the cells sequentially to reproduce the findings and visualizations.

Analysis Results
The project provides insights into compensation trends, helping to visualize the "experience premium" and identifying any notable variances in the provided data sample.
